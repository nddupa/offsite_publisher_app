# Offsite Publisher App

This is an internal prototype web app for finding relevant Reddit discussions and helping a team review potential reply opportunities.

The app is intended to search recent Reddit posts based on selected keywords, display matching threads in a review table, and help prepare draft replies that a human can edit or approve before posting.

## Intended Use

- Search recent Reddit discussions by keyword
- Display relevant threads for manual review
- Show useful context such as subreddit, title, URL, date, score, and comment count
- Allow a human reviewer to select a thread
- Generate a suggested reply draft for human editing
- Require human review and approval before posting

## Compliance Notes

This app is not intended for spam, mass posting, automated engagement, vote manipulation, scraping at high volume, or bypassing Reddit limits.

All replies are intended to be reviewed and edited by a human before posting. The app is for development and internal testing only.

## Status

This project is currently in early development.
